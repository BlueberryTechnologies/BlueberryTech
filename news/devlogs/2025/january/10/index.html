Happy new year everyone!! It's been a while since our last update but we've been hard at work with pushing updates for everyone and making sure everyone has a good experience with our services!

Recent Changes

We have been updating and maintaining the Minecraft Server play.blueberry.dev, and as of writing this the server is playable on 1.21.3=>1.21.4

We've ran into some player issues when playing on newer versions but we utilize ViaVersion to achieve the ability for players to play on a separate client version than the server.

For those of you who don't know, we host everything in house from the webpage you're reading this on, to the Minecraft Server. It was a nice challenge to get everything to work together and kind of a pain... A more in-depth explanation is that we use Ubuntu Server 24.04.1 and we use a combination of MSCS to have multiple worlds managed all together. We have all of our worlds setup with Paper and then we have a Minecraft Proxy managing all of the worlds called Waterfall. Please note that Waterfall has reached EOL, but we will try to support it as long as we can mostly because migrating everything will result in a loss of some plugins that we use. We have multiple worlds that the player can select from such as: Survival, Creative, Hardcore and optional legacy servers.

We have also added cross platform play with Java & Bedrock editions, so no matter where you are or how you play you can enjoy our server!
