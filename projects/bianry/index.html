Bianry

A stronger password generator in Python forked from Mango, it utilizes the time library to create a unique seed to generate a password. It will also have the capability to embed that password inside of an image, either a .png or a .bmp image, write to a .json file and append with existing passwords, as well as a regular .txt file.

We are also experimenting with the idea of using the entire image as a password, and possibly the ability to use a cipher to encrypt the password with a given key generated by time. Stay tuned for any updates regarding that.
