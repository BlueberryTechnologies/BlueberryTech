Our story.

Seeking a better future where collaboration and working together comes first, we prioritize being open. Learning from one another is very important, that's why all of our code is open source under GNU General Public License v3.0 which you can find more information here. This license allows us to publish code to you, and you can take that code and modify it. Whatever you end up with must have the same license, this prevents taking something we wrote and making it closed source. We want everyone to be able to learn something from someone. Nobody should be hiding information from others, we all deserve to learn.

You can find our GitHub with all of our open source code. Available for you, totally free.
