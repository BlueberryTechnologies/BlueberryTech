Documentation

Quickstart

Welcome to the Blueberry Technologies Barcode Editor. This program allows you to create and print multiple different kinds of barcodes.

The current types allowed are:

- QR Codes
- Aztec Codes
- Code 128 barcodes

Also as a bonus:
- Plain Text
- Images

Configuration

To get started you will need to have a printer configured on your computer.

This could be a wireless or wired network printer, or a wired printer.

Once a printer is setup on your device, the program should automatically recognize it and it will be available to print. ONLY ONE PRINTER IS AVAILABLE, PRINTER SELECTOR COMING SOON

You are able to change the path the program looks for generated codes. In Menu > Printer Settings you are able to change to a custom path, or revert back to default.

If you have issues selecting a directory, please make sure the selected directory is not read only.
